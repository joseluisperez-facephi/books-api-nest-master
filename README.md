# books-api

1. Clonar proyecto
2. ```yarn install ```
3. Clonar el archivo ```.env.template```y renombrarlo a ```.env ```
4. Cambiar las variables de entorno
5. Levantar la BBDD
````
docker-compose up -d
````
6. Levantar: ```yarn start:dev```


Nota: en el archivo ```tsconfig.json```hay que cambiar el target a ```es2021```
